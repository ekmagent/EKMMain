# YouTube Scripts — Recording Guide

Five long-form scripts sourced entirely from live medicareyourself.com pages. Every rate, percentage, and date in these scripts appears on the corresponding source page — if a page's data refreshes (new CSG snapshot), re-verify the figures before recording.

## Recording order recommendation

1. **yt-05-when-to-apply.md** — Record first. Evergreen, no rate tables to memorize, easiest single-take script, and it's the video every other video's end-screen should point to (everyone's next step is "check your window").
2. **yt-01-plan-g-vs-n-nj.md** — Second. Highest-intent search topic, moderate data load, and it warms you up on quoting the NJ carrier ladder you'll reuse in #4.
3. **yt-04-best-medigap-nj-rates.md** — Third. Reuses the same CSG NJ dataset as #1 while it's fresh in your head; link both videos to each other.
4. **yt-02-humana-medigap-review.md** — Fourth. Heaviest data script (four states of filings and loss ratios); by now the "loss ratio → rate filing" explanation is muscle memory.
5. **yt-03-mutual-of-omaha-medigap.md** — Last. Same report-card format as Humana, so it's the fastest of the two carrier reviews to record — and the four-entity NAIC section benefits from a confident, practiced delivery.

Batch tip: record #2 and #3 in the same session — same structure, same on-screen table style, same "adverse selection spiral" explainer with different numbers.

## Gear notes

Any modern phone camera at eye level with a $20 lavalier or USB mic is fully sufficient — audio clarity matters far more than image quality for this format. Record in a quiet room, show the on-screen data callouts as simple text/table overlays (screenshots of the actual site tables work great and reinforce the source).

## Why long-form, not Shorts

AI engines (Google AI Overviews, ChatGPT, Perplexity) cite long-form, reference-structured YouTube videos — 10+ minutes, clear chapter headings, long descriptions with timestamps — regardless of channel size. Description length and timestamp chapters matter more than subscriber count. Shorts get views; chaptered reference videos get cited. Every script here is built as a citable reference document first and a video second: keep the chapter structure intact when editing, paste the full description with timestamps, and keep the source-page URL in every description.
